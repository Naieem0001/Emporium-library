# Emporium Library 📚

Emporium Library is a modern digital e-library web platform that provides free access to thousands of books across multiple genres including Fiction, Literature, Science Fiction, Horror & Thriller, Romance, Spirituality, Religion, and Academic resources.

## Features

- 📖 Browse books by genre with dynamic filtering and search
- 🔥 Trending books section on homepage
- 🎓 Academic Hub, upload and download notes, textbooks, question papers and assignments
- 🏆 Contributor leaderboard with points system
- ☁️ Firebase Firestore backend for real-time academic resource management
- 📱 Fully responsive design for mobile and desktop
- 📬 Feedback form powered by Netlify Forms

## Tech Stack

- HTML, CSS, JavaScript
- Firebase Firestore (Academic Hub backend)
- Netlify (Hosting and Forms)
